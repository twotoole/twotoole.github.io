Yu-Gi-Oh! Fan Project

Link to scripts: github.com/twotoole/yugioh-Scripts-

I wanted to make a game using ripped assets to create a type of game fans of the franchise are unlikely to see. For this I wanted to make 'what if there was a more realistic Yu-Gi-Oh! game?'. Fairly early on it was obvious that the time it would take to implement a full Yu-Gi-Oh! duel simulation and a game that fans would actually want to and enjoy playing would be prohibitive, and the effect from seeing a slice of gameplay in a video/trailer would evoke a simliar response.

Technically this was a chance to use polymorphism with scriptable objects for the different card types.
