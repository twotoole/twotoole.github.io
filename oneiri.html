O-Neir-I

Link to scripts: github.com/twotoole/ONeiri-Scripts-

This was a 3 person team project for a university module. We ran this project using agile development with 2 week sprints where tasks were maintained using Trello boards. The goal was to make a puzzle platformer with extra attention to design in a way that guides the player while limiting direct instructions.


My main contributions were in:
  • Gameplay scripting
  • Running and documenting playtest sessions
  • Level prototyping
