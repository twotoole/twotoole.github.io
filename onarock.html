On A Rock

Link to scripts: github.com/twotoole/OnARock-Scripts-

This was a game for a univeristy module. The goal was to make a 3D platformer (N64 style) with an environment generated by using photogrammetry and images from Google maps terrain. While solely relying on this didn't work for creating a compelling game world, I've been able to use the technique to generate assets for other projects.
